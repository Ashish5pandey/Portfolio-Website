Project

E-Commerce Product Catalog Web Application

Project Overview:
During my training at Wipro, I successfully developed an E-Commerce Product Catalog Web Application using Spring Boot and Thymeleaf.
This project is a robust and user-friendly e-commerce web application developed using Spring Boot and Thymeleaf. Designed to manage a product catalog seamlessly, the application caters to both customers and admin users, offering an intuitive interface for viewing and managing products.

Features:
Home Page: Displays a list of products with names, prices, and descriptions.
Product Details Page: Shows detailed information about each product.
Admin Controls: Admins can add, edit, and delete products seamlessly.

Technologies Used:
Backend: Spring Boot, Spring Data JPA, RESTful APIs
Frontend: Thymeleaf, HTML, CSS, Bootstrap
Database: SQL, MySQL
Tools: Postman, Maven, IntelliJ IDEA

My Contributions:
* Designed and developed RESTful APIs using Spring Boot for E-Commerce data management.
* Implemented dynamic web pages using Thymeleaf for a responsive user interface.
* Integrated with MySQL for database operations, ensuring data consistency and integrity.
* Conducted rigorous testing with Postman to validate API functionality.

What I Learned:
This project enhanced my full-stack development skills, deepened my understanding of Java Spring Boot, and strengthened my ability to design intuitive user interfaces with Thymeleaf. Project Deployment acquired experience in deploying web applications in various environments.
